# 📁 04. リポジトリ構造

このプロジェクトのディレクトリ構成と、主要なファイルの役割を説明します。

## 📂 ルートディレクトリ

```text
memo-platform/
├── auth-api/          # バックエンド (Express / Prisma)
├── my-login-app/      # フロントエンド (React / Tailwind)
├── wiki/              # プロジェクトドキュメント (このドキュメント)
└── ドキュメント指示書.md # ドキュメント生成ガイド
```

## 🚀 バックエンド (`auth-api/`)

```text
auth-api/
├── bin/
│   └── run_server.ts  # エントリーポイント
├── prisma/
│   ├── schema.prisma   # データモデル定義
│   └── seed.ts         # 初期データ投入スクリプト
├── src/
│   ├── app.ts          # Express アプリ構成
│   ├── controllers/    # リクエストハンドラ (REST エンドポイント)
│   ├── services/       # ビジネスロジック
│   ├── models/         # データ操作 (DAO 相当)
│   ├── middleware/     # 認証・バリデーションミドルウェア
│   ├── routes/         # ルーティング定義
│   └── utils/          # 共通ユーティリティ
└── package.json        # 依存関係とスクリプト
```

## ⚛️ フロントエンド (`my-login-app/`)

```text
my-login-app/
├── public/             # 静的アセット
├── src/
│   ├── components/     # 再利用可能なReactコンポーネント
│   ├── hooks/          # カスタムHooks
│   ├── App.tsx         # メインアプリケーションコンポーネント
│   ├── index.tsx       # エントリーポイント
│   └── index.css       # グローバルスタイル (Tailwind読み込み)
├── tailwind.config.js  # Tailwind CSS設定
└── package.json        # 依存関係とスクリプト
```
