# 📄 01. プロジェクト概要

## 🌟 プロジェクトについて

**Memo Platform** は、ユーザーが自分専用のメモを作成・管理できるフルスタック Web アプリケーションです。
シンプルかつ強力な認証機能を備え、プライベートな情報を安全に保存することを目的としています。

## 🎯 主な目標

- **セキュリティ**: JWT を用いた堅牢なユーザー認証。
- **使いやすさ**: React と Tailwind CSS による直感的な UI。
- **データ管理**: Prisma ORM を使用した信頼性の高いデータ操作。

## 🛠 テクノロジースタック

### バックエンド (`auth-api`)

- **Node.js / TypeScript**
- **Express**: 軽量で拡張性の高い Web フレームワーク。
- **Prisma**: 型安全なデータベース操作を実現する ORM。
- **MySQL**: 信頼性の高いリレーショナルデータベース。

### フロントエンド (`my-login-app`)

- **React (v19)**: 最新の React 機能（React Hook Form, Zod 等）を活用。
- **Tailwind CSS**: 高速なスタイリングとレスポンシブデザイン。
- **TypeScript**: コードの品質と保守性を向上。

## 💡 主要機能

- **ユーザー登録 & ログイン**: 安全なパスワードハッシュ化とセッション管理。
- **メモ管理**: 作成(C)、取得(R)、更新(U)、削除(D)のフル機能。
- **レスポンシブ UI**: デバイスを問わず快適に使用可能。
