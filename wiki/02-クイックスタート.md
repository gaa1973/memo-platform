# 🚀 02. クイックスタート

このセクションでは、プロジェクトをローカル環境でセットアップし、実行する手順を説明します。

## 📋 前提条件

以下のツールがインストールされていることを確認してください：

- **Node.js**: v18 以上 (v20 推奨)
- **MySQL**: (バックエンドで使用)
- **npm**: (Node.js に含まれています)

## 🛠 セットアップ手順

### 1. リポジトリのクローンと依存関係のインストール

```bash
# プロジェクトルートに移動
cd memo-platform

# バックエンドのセットアップ
cd auth-api
npm install

# フロントエンドのセットアップ
cd ../my-login-app
npm install
```

### 2. 環境変数の設定

`auth-api` ディレクトリに `.env` ファイルを作成し、データベース接続情報を設定します。

```env
# auth-api/.env
DATABASE_URL="mysql://user:password@localhost:3306/memo_db"
JWT_SECRET="your_secure_secret_key"
```

### 3. データベースの初期化 (Prisma)

```bash
cd auth-api
# マイグレーションの実行
npm run db:migrate
# シードデータの投入
npm run db:seed
```

### 4. アプリケーションの起動

#### バックエンドの起動

```bash
cd auth-api
npm start
# サーバーが http://localhost:8000 で起動します
```

#### フロントエンドの起動

```bash
cd my-login-app
npm start
# ブラウザが http://localhost:3000 で自動的に開きます
```
